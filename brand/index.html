Brand · iteration 2 (Gemini track)

Side-by-side: Claude vs Gemini

Two parallel brand kits. Each row shows the same artifact rendered from each track. The palette is shared — the difference is the mark execution and the supporting hero imagery.

Iteration 2 update (Gemini track). The 2D mark has been re-vectorized end-to-end via the free open-source stack: Gemini's 2D panel extracted with precise color-mask bounding, LANCZOS-upscaled 6×, then traced with VTracer in binary mode. No more hand-tracing from a low-res JPG. The Gemini geometry (outer ring, offset inner bowl, ascender, bottom-right design detail) is now faithfully preserved. The 3D hero is still Gemini's native render matted on stone.
Open Claude brand book → Open Gemini brand book → Original Gemini guide (source JPG)
Open tradeoffs. Gemini's wordmark lockup and mark are now vector-accurate to the source guide. The 3D hero stays Gemini's original render (cropped + matted on native stone); the transparent cut still has some shadow loss. Claude's mark remains its own geometric concept. Favicon at 16px is cramped on the Gemini side — a simplified "small-size" favicon variant would help if the mark wins.

Where this goes next

If Claude wins: the SVG geometry is solid (musical-note risk avoided), faviconizes cleanly at 16px, but there's no 3D hero asset. We'd build one.
If Gemini wins: the 3D hero is the standout — nothing on the Claude side touches it. The 2D vector trace needs another pass to nail the bowl/ascender geometry from a higher-resolution source.

The site root currently points to /brand/claude/... for favicons and OG. Once you pick, swap one path and we're done.
